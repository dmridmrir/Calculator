# Calculator
학교 github 프로젝트 실습

## Goals

* github 장소 생성 및 관리 실습
* PyQt5를 이용한 계산기 기능 및 사용방법 개선
* 실무에서 github를 사용할 정도의 실력 구비
  
## Getting started

* PyQt5 모듈 설치 (pip install PyQt5)
* 계산기 파일 실행 (python calaculator_main.py)
